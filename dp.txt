動的計画法
元の大元の問題を部分問題に分割して解いていく方式
漸化式を作ってどんどん解いていくイメージ
大体漸化式の評価の中にmax, minを判別するようなものがある
各部分問題でどっちが大きいか, どっちが小さいか選んでいく
各漸化式が部分問題に相当する


例)最大重み区間問題

-2 4 -1 4 -5 6 -12 3
↑
||
  ↑
  ||
    ↑
  |  |
       ↑
  |     |
          ↑
  |     |   
            ↑
  |          |
               ↑
  |          |
                 ↑
  |          |

「途中まで重み最大区間が決まっているとすると
その区間の合計と, 今見ている数値を右端として重み最大の区間の合計の中で一番大きい物を比較する
そのうち大きい方を現在の重み最大区間とする」
という部分問題を解いていくことで元の問題を解く.

W[i]は区間{1,i}の重み最大の区間
W_right[i]は右端がiのうち重み最大の区間
W_right[i + 1] = max{a[i+1}, W_right[i] + a[i+1]}
W[i + 1] = max{W[i], W_right[i + 1]}
